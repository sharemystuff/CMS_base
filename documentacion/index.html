Núcleo del Sistema

Funciones fundamentales que rigen el comportamiento de la plataforma y el entorno.

url_base()api/main.php

Calcula dinámicamente la URL absoluta del sitio. Detecta automáticamente si el servidor usa HTTPS (vía mkcert o producción) y resuelve puertos no estándar.

Params: Ninguno
Retorna: string (ej: https://www.adminbase.mahg)
asset()api/main.php

Función vital para cargar CSS, JS e Imágenes. Concatena la URL base con el path proporcionado y añade un timestamp opcional para evitar el caché del navegador en desarrollo.

Params: $path (string)
Retorna: string (URL absoluta del recurso)
checking()api/main.phpAUTH

Verifica la existencia de una sesión válida. Comprueba que el ID de usuario y el token de sesión coincidan con la base de datos.

Retorna: bool

Seguridad (El Escudo)

Funciones encargadas de neutralizar ataques XSS, SQLi y CSRF.

limpiar_entrada()seguridad/funciones.php

El primer filtro. Realiza trim(), stripslashes() y htmlspecialchars() para desinfectar cualquier dato proveniente del usuario.

Params: $data
e()seguridad/funciones.php

Escapado de salida. Debe usarse en cada 'echo' dentro de etiquetas HTML para prevenir Cross-Site Scripting.

Retorna: string (escapado)
validarCSRF()api/main.phpCRÍTICO

Verifica que el token enviado en formularios POST coincida con el secreto de la sesión para evitar peticiones maliciosas externas.

Retorna: bool
email_valido()seguridad/funciones.php

Usa filter_var con FILTER_VALIDATE_EMAIL tras limpiar caracteres ilegales.

Retorna: bool

Usuarios & Autenticación

Lógica de manejo de cuentas, recuperación y contraseñas.

crear_usuario()api/funciones_model.phpINSERT

Registra un usuario. Aplica el hasheo de contraseña automáticamente antes de insertar.

Params: $nombre, $nick, $email, $pass, $rol
encode_pass()api/funciones_model.php

Aplica PASSWORD_BCRYPT con un 'cost' optimizado para la seguridad actual.

Retorna: string (hash 60 chars)
generar_token_recuperacion()api/funciones_model.php

Genera un token hexadecimal y establece 'reset_expira' a +5 minutos en la DB.

Retorna: string (token)
validar_token_recuperacion()api/funciones_model.php

Comprueba si el token existe y si el tiempo actual no ha superado el límite de 5 minutos.

Retorna: array (datos user) | false
get_user()api/funciones_model.php

Obtiene toda la información de un usuario por su ID.

Retorna: object/array
actualizar_ultimo_login()api/funciones_model.php

Actualiza el timestamp 'ultimo_login' y la IP del usuario al entrar al panel.

Contenidos (Posts & Metas)

El motor de contenido del CMS.

get_post()api/funciones_model.php

Recupera un post o página filtrando por ID o por URL (slug). Soporta carga de metadatos adjuntos.

crear_post()api/funciones_model.php

Inserta contenido en la tabla 'posts'. Gestiona automáticamente la creación del slug si no se proporciona.

update_post_meta()api/funciones_model.php

Guarda o actualiza un valor dinámico en la tabla 'post_meta'. Fundamental para SEO y Custom Fields.

get_post_meta()api/funciones_model.php

Recupera un valor específico de metadato para un post concreto.

Opciones Globales

Configuración del sitio y constantes de base de datos.

get_opcion()tovi/funciones.php

Recupera un valor de la tabla 'opciones'. Es la función más usada para configurar el nombre del sitio, SMTP, etc.

Retorna: string | null
update_opcion()tovi/funciones.php

Actualiza una opción existente o la crea si no existía.

get_all_opciones()tovi/funciones.php

Carga todas las opciones en un único array asociativo para reducir consultas SQL al servidor.

Utilidades de Pelín

Herramientas de desarrollo, debug y generación de datos.

pre()tovi/funciones.phpDEBUG

Envuelve un print_r entre etiquetas <pre> con estilos CSS para facilitar la lectura de objetos y arrays complejos.

random_pass()tovi/funciones.php

Genera una cadena de texto aleatoria. Útil para contraseñas temporales o sales.

enviar_email()api/main.php

Wrapper sobre PHPMailer. Configura automáticamente el servidor SMTP usando los datos de 'get_opcion'.

log_error()seguridad/funciones.php

Escribe eventos críticos en el archivo php_errors.log sin mostrar información sensible al usuario final.

Referencia Completa

Listado técnico integral de todas las funciones del sistema.

1. Seguridad y Blindaje

limpiar_entrada($data)seguridad/funciones.php

Limpia espacios, quita barras invertidas y convierte caracteres especiales. El primer filtro de todo $_POST.

e($string)seguridad/funciones.php

Escapa variables para salida HTML segura (prevención de ataques XSS).

validarCSRF()api/main.php

Compara el token del formulario con el de la sesión para evitar peticiones falsificadas.

generarCSRF()api/main.php

Crea un token único de seguridad si no existe en la sesión actual.

email_valido($email)seguridad/funciones.php

Valida el formato legal y sanitiza correos electrónicos.

log_error($mensaje)seguridad/funciones.php

Registra incidentes técnicos en php_errors.log incluyendo la IP del usuario y marca de tiempo.

bloquear_ip($ip)seguridad/funciones.php

Añade una dirección IP a la "lista negra" tras detectar intentos de acceso fallidos reiterados.

es_ip_bloqueada($ip)seguridad/funciones.php

Verifica si el visitante actual tiene el acceso restringido por seguridad.

2. Núcleo y Rutas

url_base()api/main.php

Detecta dinámicamente el protocolo (HTTP/S) y la URL raíz del servidor virtual.

asset($path)api/main.php

Genera la ruta absoluta a archivos estáticos en el directorio /assets.

checking()api/main.php

Middleware de protección: verifica la sesión activa antes de cargar páginas privadas.

redirigir($url)api/main.php

Maneja salidas limpias y redirecciones HTTP seguras.

enviar_email($to, $subject, $body)api/main.php

Orquestador de envíos vía PHPMailer utilizando la configuración SMTP de la DB.

3. Modelos de Datos

crear_usuario(...)api/funciones_model.php

Registro de nuevos perfiles en la base de datos.

encode_pass($password)api/funciones_model.php

generar_token_recuperacion($email)api/funciones_model.php

Crea una 'special_key' única con una validez estricta de 5 minutos.

validar_token_recuperacion($token)api/funciones_model.php

update_post_meta($id, $key, $val)api/funciones_model.php

Guarda campos personalizados utilizando una arquitectura EAV.

get_post($id_or_slug)api/funciones_model.php

Motor de búsqueda de contenidos por ID o URL amigable.
